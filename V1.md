## 🌧️ Script ejemplo (estructura)

```
scripts/02_clima.R
```

```r
library(sf)
library(dplyr)

mun <- st_read("output/municipios_base.geojson")

centroides <- st_centroid(mun)

# Extraer coordenadas
coords <- st_coordinates(centroides)

mun <- mun %>%
  mutate(
    lon = coords[,1],
    lat = coords[,2]
  )

# Aquí se añadirán:
# temperatura_invierno
# precipitacion

# (placeholder manual)
mun <- mun %>%
  mutate(
    temp_invierno = runif(n(), 2, 8),
    precipitacion = runif(n(), 300, 800)
  )

st_write(mun, "output/municipios_clima.geojson", delete_dsn = TRUE)
```

## 🚗 4. Accesibilidad (fase inicial)

### Objetivo mínimo:

* Distancia aproximada a Madrid

### Estrategia simple (v1):

```r
madrid <- c(-3.7038, 40.4168)

mun <- mun %>%
  mutate(
    dist_madrid = geosphere::distHaversine(
      cbind(lon, lat),
      madrid
    ) / 1000  # km
  )
```

👉 Esto NO es tiempo real, pero sirve como aproximación inicial.
