# Cómo leer los resultados con criterio

Esta página resume cómo revisar el ranking sin confundir un indicador útil con una verdad absoluta.

Uno de los objetivos principales del Atlas es reducir el peso de prejuicios, clichés o impresiones previas sobre el territorio, y acercarse a una evaluación lo más objetiva posible basada en datos comparables.

Sirve para compartir resultados con terceros de forma honesta: mostrando fortalezas y límites.

## La regla principal

El Atlas no dice "este lugar es mejor". Dice "con estas reglas y este alcance, este lugar queda mejor situado que otros".

Esa diferencia importa. Permite usar el Atlas como una herramienta de conversación y priorización, no como una autoridad cerrada.

## Qué significa "objetividad" aquí

En este Atlas, objetividad significa algo concreto: reglas visibles, cálculo reproducible y trazabilidad.

No significa neutralidad perfecta ni ausencia de decisiones metodológicas. Los pesos y umbrales existen, y están documentados para poder revisarlos.

Por eso, una lectura honesta debe mencionar siempre el criterio usado: qué alcance, qué pesos, qué indicadores y qué límites.

## Preguntas de robustez útiles

Para evaluar si el ranking aguanta, conviene revisar:

1. Si cambios razonables de pesos alteran mucho el orden.
2. Si el patrón espacial tiene sentido con la geografía observada.
3. Si cada indicador se interpreta dentro de su alcance real.
4. Si el resultado coincide con una señal amplia del territorio o depende de un dato aislado.
5. Si hay una explicación clara para los municipios que sorprenden.

## Cómo detectar una lectura débil

Hay señales de que una interpretación necesita más trabajo:

- Se usa solo el ranking, sin mirar bloques.
- Se presenta un score alto como argumento definitivo.
- Se ignora que el score es comparativo dentro del alcance actual.
- Se confunde acceso fluvial recreativo con calidad sanitaria del agua.
- Se da por buena una conexión de transporte sin revisar distancia, frecuencia o acceso real.

## Checklist de confianza

- Se puede trazar cada métrica a su script y fuente.
- La ejecución se puede reproducir con comandos documentados.
- Los límites metodológicos están explícitos.
- Cambios de pesos o umbrales quedan registrados.
- El resultado tabular y el geoespacial son consistentes.
- Las conclusiones se explican con palabras, no solo con números.

## Qué no conviene hacer

No conviene usar el score como argumento definitivo ni como herramienta para presionar decisiones dentro del grupo.

El Atlas orienta el análisis y aporta información comparable, pero no sustituye el diálogo, el criterio compartido ni la construcción colectiva de la decisión.

Una buena lectura debería acabar con una lista corta de lugares a revisar, no con una decisión cerrada solo por datos.
