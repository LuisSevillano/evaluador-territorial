# Release y despliegue

## Flujo

1. Regenerar dataset y tiles.
2. Ejecutar QA técnico/metodologico.
3. Build frontend y build docs.
4. Desplegar con `npm run deploy` en `frontend/`.

## Control de cambios

- Registrar cambios de umbrales y pesos.
- Registrar cambios de scope analitico.
- Registrar nuevas fuentes y fallback usados.
