# Diccionario de indicadores

Está sección centraliza definiciones, fórmulas, rangos, fuentes y limitaciones.

Para no perderse en la nomenclatura, conviene leer por capas: primero clima, después accesibilidad, luego naturaleza y ríos, y al final el score compuesto donde todo converge.

## Bloques

- [Clima](/indicadores/clima)
- [Accesibilidad](/indicadores/accesibilidad)
- [Naturaleza y ríos](/indicadores/naturaleza-rios)
- [Score compuesto](/indicadores/score)

## Convenciones

- Variables normalizadas: sufijo `_norm` en rango `[0,1]`.
- Variables en porcentaje: sufijo `_pct`.
- Variables de clasificación: sufijo `_class`.
